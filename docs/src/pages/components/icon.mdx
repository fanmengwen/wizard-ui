语义化矢量图标。通过 [icomoon](https://icomoon.io) 将 SVG 转成字体图标得到。

## 使用场景
直观展示某个资源或者某种状态。

## 代码演示
```jsx
<Icon type="os-search-role"/>
```

## API
```jsx previewOnly
<PropTable of="icon" />
```

## 所有图标
```jsx previewOnly
<AllIcon/>
```
