文字水井展示。

## 使用场景

适用于文字增加间隔提示信息。

## 基本用法

```jsx
<div>
  <Well>this is well</Well>
</div>
```

## 不同大小间距

```jsx
<div>
  <Well size="large">this is well</Well>
  <Well size="sm">this is well</Well>
</div>
```
