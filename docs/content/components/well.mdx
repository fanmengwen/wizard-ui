按钮用于开始一个即时操作，触发业务逻辑时使用。

## 使用场景

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

## 基本用法

```jsx
<div>
  <Well>this is well</Well>
</div>
```

## 不同大小间距

```jsx
<div>
  <Well size="large">this is well</Well>
  <Well size="sm">this is well</Well>
</div>
```
